ArthDrishti — Project File Structure
=====================================
AI-powered business intelligence platform. React 18 + Vite SPA frontend,
Supabase PostgreSQL + Deno edge functions backend, Groq LLaMA 70B inference.

.
├── api/
│   └── analysis.ts               Vercel serverless function: injects SSR Open Graph / Twitter Card
│                                 meta tags into dist/index.html for /analysis/:id social sharing
│
├── public/
│   ├── favicon.ico               Browser tab icon
│   ├── manifest.json             PWA manifest: app name, icons, display mode, theme color
│   ├── robots.txt                Search engine crawl directives
│   └── sitemap.xml               XML sitemap for SEO (lists public routes)
│
├── src/

│   ├── index.css                 Global Tailwind CSS directives, CSS variables for theme tokens,
│   │                             custom utility classes (safe-area insets, animations)

│   │   │                         featured analysis card

│   │   │                         meta tags, increments view count, renders AnalysisResults
│   │   ├── BusinessOfDay.tsx     /botd — Daily Business of the Day: fetches today's pre-generated

│   │   ├── Simulator.tsx         /simulator — Decision quiz game: 3-option multiple choice drawn
│   │   │                         from real decision data in cached analyses

│       ├── analyze-business/

│       │                         rate limits, fetches Wikipedia context, calls Groq LLaMA
│       │                         3.3 70B with structured JSON schema, returns analysis
│       ├── generate-botd/
│       │   └── index.ts          Cron-triggered: deterministically selects daily business,

│       ├── verify-razorpay-payment/
│       │   └── index.ts          Verifies HMAC-SHA256 payment signature, inserts pro
